## Задание 2. Формирование нового сводного файла текущего года
## Выполняли студенты группы М3О-134БВ-25 Молоканова Анастасия и Шихобалов Илья

Программа предназначена для создания нового сводного файла текущего учебного года на основе двух исходных Excel-файлов:
`Список 15вар - осень.xls` и `Список 15вар - весна.xls`

В новый файл переносятся данные из осеннего и весеннего семестров. Для каждой строки дополнительно добавляется столбец `Семестр`, чтобы было понятно, из какого файла взяты данные.

## Что делает программа
1. Загружает файл с нагрузкой за осенний семестр.
2. Загружает файл с нагрузкой за весенний семестр.
3. Добавляет в таблицу осени значение `Осень`.
4. Добавляет в таблицу весны значение `Весна`.
5. Объединяет две таблицы в одну.
6. Сохраняет результат в новый Excel-файл:

`Сводный файл текущего года.xlsx`

## Используемые библиотеки
Для работы программы нужны библиотеки:
pandas
openpyxl
xlrd

Установить их можно командой:
```bash
python -m pip install pandas openpyxl xlrd
```

## Код программы

```python
import pandas as pd

osen = pd.read_excel("Список 15вар - осень.xls")
vesna = pd.read_excel("Список 15вар - весна.xls")

osen["Семестр"] = "Осень"
vesna["Семестр"] = "Весна"

itog = pd.concat([osen, vesna], ignore_index=True)

itog.to_excel("Сводный файл текущего года.xlsx", index=False)

print("Файл создан")
```

## Результат работы

В результате выполнения программы создается новый сводный Excel-файл, в котором объединены данные осеннего и весеннего семестров текущего учебного года.
